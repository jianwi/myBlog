## 介绍
这是一个基于Vue3的博客系统。  
后台用 Koa，各种数据全都存到语雀里  
## 愿景
以最最简单的方式实现一个功能强大的博客系统。    
不需要后台管理系统，全都基于语雀。  

# 版本计划
### v 1  先做出来
> 一期的目标很简单，实现显示博客列表，点击跳转到语雀  
- 写死的博客首页
- 显示列表和跳转

### v 2  简单的管理功能
- 支持标签  
- 支持配置首页标题，头像，介绍等文字  

### v3   实现可迁移
让别人也能很方便的使用
